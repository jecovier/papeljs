Partials Pages

When working with PapelJS, you can create partial pages that will be loaded into the slots content areas of your layout. This allows you to create reusable designs that can be used across multiple pages.

all partial pages needs to have the papel.js file included in the body of the file and the pl-layout attribute pointing to the layout file to render. This is required to render the layouts of the page when accessing the partial page directly in the browser.

Named Slots

You can create named slots by adding the name attribute to any slot in your layout. This will create a named slot where you can load partial contents with the same name.
