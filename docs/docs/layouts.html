Layouts

By default your index.html file will be the main layout of your website. This is where the partial pages will be loaded. you can define different layouts for different pages by adding the pl-layout attribute. but your index.html file will always be the main layout of your website.

To add layouts capabilities to your file, you need to create slots where the partial pages will be loaded. You can create as many slots as you need and load different partial pages into each one. But there will be at least one slot in your index.html file, which is the main content. this is where the partial pages will be loaded by default and it doesn't requires a name.

Named Slots

You can create named slots by adding the name attribute to any slot in your layout. This will create a named slot where you can load partial contents with the same name.
