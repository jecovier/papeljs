Get started

trazojs is a lightweight and easy-to-use JavaScript library that lets you manage layouts and partial pages using just HTML files. It runs entirely on the client side, so no server is needed.

Installation

To start using trazojs, you can choose one of the following methods:

1. Download and Include Locally Download the latest version of trazojs and add it to your project. Then include it in your index.html file:

2. Use the CDN For a quicker setup, include trazojs directly from the CDN: https://cdn.jsdelivr.net/gh/jecovier/trazojs/dist/trazos.js
