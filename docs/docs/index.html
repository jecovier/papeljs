Get started

PapelJS is a simple, lightweight JavaScript library that allows you to work with layouts and partial pages using only html files. It is designed to work on the client side and does not require a server to run.

Installation

To get started, download the latest PapelJS file and add it into your index.html:

Alternatively, you can use the CDN version: https://cdn.jsdelivr.net/gh/jecovier/papeljs/dist/papel.js
