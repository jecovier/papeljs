Navigation Boost

by default PapelJS will fully render normal and partial pages. To improve the navigation experience, you can use the pl-boost attribute in links to load only the partial content of the page. This will perform an xhr call to the partial page and load only the content of the page into the slots.

Prefetching

To improve the loading time of the pages, you can use the pl-prefetch to load the partial page into the cache of the browser. This will add an event listener to the link and load the partial page into the cache as soon as the link is visible in the viewport.

Prefetchin "ALL"

If you want to prefetch all the images in the next partial page, you can use the pl-prefetch="all" attribute. This will preload the next partial page, get all the images in there and start loading them into the cache.
