Documentation

papeljs is a lightweight and easy-to-use JavaScript framework for building modern web applications. This documentation will guide you through all the features and concepts.

Installation

To start using papeljs, you can choose one of the following methods:

1. Download the file

Download papeljs and add it to your project: 

<script src="papeljs.js"></script>

2. Use the CDN

For a quicker setup, include papeljs directly from our CDN:

<script src="https://cdn.jsdelivr.net/gh/jecovier/papeljs/dist/papeljs.js"></script>

Quick Start

Here's a simple example to get you started:

<!DOCTYPE html>
<html>
<head>
    <title>My App</title>
    <script src="https://cdn.jsdelivr.net/gh/jecovier/papeljs/dist/papeljs.js"></script>
</head>
<body>
    <h1>Hello World!</h1>
</body>
</html>
