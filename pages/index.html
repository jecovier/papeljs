Getting Started

Step 1 - Add papeljs to your index.html

<script src="https://cdn.jsdelivr.net/gh/jecovier/papeljs/dist/papeljs.js"></script>

Step 2 - Create your first page

<!DOCTYPE html>
<html>
<head>
    <title>My App</title>
</head>
<body>
    <h1>Hello World!</h1>
    <script src="https://cdn.jsdelivr.net/gh/jecovier/papeljs/dist/papeljs.js"></script>
</body>
</html>

Step 3 - Enjoy the simplicity

That's it! You now have a fully functional web application included in papeljs and enjoy the simplicity of creating fast and responsive web applications.
